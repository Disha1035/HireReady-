# 🎯 HireReady - AI Mock Interview Platform

HireReady is an AI-powered mock interview platform designed to help students prepare for technical and HR interviews with confidence. The platform simulates a real interview experience by asking questions orally, accepting voice or text responses, providing instant feedback, and tracking interview performance.

## 🚀 Features

- 🤖 AI-based HR and Technical Interview Simulation
- 🎤 Voice-based Interview Questions
- ⌨️ Text and Speech-to-Text Answer Support
- ⏱️ Built-in Interview Timer
- 📊 Real-time Performance Tracking
- 📝 Instant Feedback System
- 🎯 Difficulty Levels (Easy, Medium, Hard)
- 📈 Progress Tracking Dashboard
- 🔐 Login and Authentication System
- 🔑 Forgot Password Feature
- 💾 Remember Me Functionality
- 📱 Responsive User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Web Speech API
- Speech Recognition API
- Local Storage


## 🎯 How It Works

1. User logs into the platform.
2.Select the comapany.
3.Select the Activity(Apttitude ,Coding ,Mock ,HR ,Technical Interview.
4. AI interviewer asks questions.
5. User answers using voice or text.
6. AI provides instant feedback.
7. Timer tracks interview duration.
8. Final performance report is displayed.
